Роль: Ты — эксперт по фронтенд-разработке со специализацией на создании сложных и эстетичных пользовательских интерфейсов. Твоя задача — в точности воссоздать визуальный эффект интерфейса операционной системы macOS с помощью CSS.

Задача: Напиши HTML и CSS код для модального окна (попапа), фон которого полностью имитирует эффект "Vibrancy" (полупрозрачное размытое стекло) из macOS.

Ключевые требования к визуальному эффекту:

Структура HTML:

Создай родительский блок-оверлей (<div class="macos-popup-overlay">), который будет занимать весь экран.

Внутри него размести блок для контента (<div class="macos-popup-content">).

Стили для оверлея (эффект размытия):

Позиционирование: Оверлей должен быть зафиксирован и растянут на весь экран (position: fixed, top, left, width, height).

Размытие фона (Backdrop Filter): Это самый важный элемент. Используй свойство backdrop-filter: blur() для размытия всего, что находится позади оверлея. Установи значительный радиус размытия, например, 18px. Обязательно добавь префикс -webkit-backdrop-filter для поддержки браузера Safari.

Полупрозрачная заливка: Чтобы создать ощущение "матового стекла", добавь легкую полупрозрачную заливку поверх размытия с помощью background-color. Предусмотри два варианта, как в macOS:

Светлая тема: background-color: rgba(255, 255, 255, 0.15); (очень легкий, почти белый оттенок).

Темная тема: background-color: rgba(20, 20, 20, 0.25); (глубокий, темный оттенок).

В итоговом коде предоставь стили для обоих вариантов, например, с помощью классов .light-theme и .dark-theme.

Центрирование: Используй Flexbox (display: flex, align-items, justify-content) для идеального центрирования контентного блока внутри оверлея.

Стили для контентного блока (само "окно"):

Фон: Контентный блок не должен иметь собственного эффекта backdrop-filter. Он должен иметь свой, почти непрозрачный фон, чтобы текст внутри был читаемым. Например, background-color: rgba(40, 40, 40, 0.6); для темной темы.

Скругленные углы: Установи border-radius в диапазоне 12px - 16px, чтобы углы были плавными, как у окон в macOS.

Граница (Edge): Добавь очень тонкую, едва заметную светлую границу, чтобы отделить окно от размытого фона. border: 1px solid rgba(255, 255, 255, 0.1);. Это создает ощущение объема.

Тень: Примени мягкую, рассеянную тень с помощью box-shadow, чтобы окно "парило" над фоном. Например: box-shadow: 0 10px 30px rgba(0, 0, 0, 0.2);.

Внутренние отступы (Padding): Установи достаточные отступы (padding), чтобы контент не прилипал к краям окна.

Формат вывода:

Предоставь единый, готовый к использованию HTML-файл.

Весь CSS должен находиться внутри тега <style> в <head>.

Добавь подробные комментарии в CSS, объясняя, за что отвечает каждое ключевое свойство (backdrop-filter, rgba, z-index и т.д.).

В HTML добавь пример простого контента внутри попапа (заголовок и параграф).

Дополнительное указание: В комментарии к CSS упомяни о кросс-браузерной совместимости backdrop-filter и о том, что для старых браузеров этот эффект не будет работать (они увидят только полупрозрачную заливку без размытия).
